# AI 编程的碎碎念

## IDE 与模型：工具与源头

首先是要分清 **IDE** 和 **模型** 是两个东西：
- **IDE**：相当于一个写代码的多功能记事本。
- **AI 模型**：则是真正输出代码的源头。

---

## IDE 的两类流派

IDE 里面又分成两类：

### 1. GUI 类（仿 VS Code 界面）
这是最常见的，如 **VS Code**、**Cursor**，以及国内的 **Trae** 等。
- **特征**：有一个 AI 问答侧边栏。
- **工作流**：通过“提问-生成-审阅-确认”来执行。
- **优点**：操作较为简单，上手快。

### 2. TUI/终端类（Agentic 风格）
最近一年最火的是这一类，代表作是 **Claude Code** (简称 cc) 和类似项目，比如 **OpenCode** (开源项目)、国内的 **Kimi Code** 等。
- **特征**：使用 TUI 终端页面，更“硬核”。
- **优势**：
    - 模型拥有更多权限。
    - **Claude Code** 会对自家 Claude 模型进行针对性的提示词和架构优化适配（同样的 OpenCode 也有各种开源优化加持）。
    - 相比第一类，更能解决系统级问题，拥有更长上下文和强大的 orchestration（编排）能力。
- **门槛**：要求使用者具备使用各种 trick 和编写大段提示词的能力，使用复杂，但效果极强。

---

## 如何“驱动”这些模型？

如何使用模型呢？
1. **获取 API Key**：去到各个模型的官网，注册登录，进入控制台拿到 API Key。
2. **配置 IDE**：把 Key 放到这些 IDE 的配置里面，就能在 IDE 里面使用模型了。

当然，这些 IDE 本身也会提供 AI 服务（集成的订阅制），比如 VS Code 的 **Copilot** 等，也可以直接登录充值使用。

## 国外模型“三剑客”对比

国内模型资料多，这里不赘述。而国外模型目前最强的只有三家，也只推荐使用这三家的模型：

### 1. Google Gemini (3 Pro/Flash)
**聊天首选，有谷歌搜索加持。**
- 内容翔实，语言风格较为理性克制。
- 在 Google AI Studio 里面可以无限免费体验最强模型。
- 当下最强无可置疑，给到夯。

### 2. OpenAI ChatGPT
**启蒙之作，普及功臣。**
- **现状**：一般，但“瘦死的骆驼比马大”，其官方 IDE **Codex** 编程很强。
- **缺点**：相较于另外两家，缺乏特色（网页美学设计还不错），回复有时显得空洞乏味。如果不个性化系统提示词（System Prompt）基本用不了。额度少，无法高强度使用，体感上甚至不如国内一些模型，越来越“NPC”，日常基本弃用。

### 3. Anthropic Claude 系列
**编程最佳选择，地表最强。**
- 各种任务轻松胜任，编程王中王，几乎能完成所有电脑任务。
- **性能**：**Claude 4.6 Opus** 毫无疑问是目前的最强梯队。无论是写日常文章还是代码，建议都使用最好的产品，节省宝贵的时间和精力，而不是在 Illusion（幻觉）里面反复。
- **局限**：公司极其“繁华”（风控严），国内 IP 见一个封一个。除非肉身出国，否则根本无法正常注册和使用，这点比不上 OpenAI。
