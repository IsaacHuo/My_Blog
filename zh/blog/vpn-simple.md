# 机场选择使用简要指南

> 这份指南只保留软件下载、订阅导入和连接步骤。

## 1. 软件下载与安装

推荐使用 **Clash Verge** (Windows/Mac) 或 **Karing** (iOS 美区)。

*   **软件下载地址**：[下载页面](https://xiazai2.yfjc.my/1)
*   **iOS 用户**：推荐使用美区 Apple ID 下载 **Karing**。
*   **Android 用户**：选择很多，请在网盘内自行选择。

以上链接包含推荐码。第三方服务可能调整价格、节点和隐私政策，先购买短期套餐测试，避免一次充值过多。

## 3. 极简使用教程

详细教程可在机场官网自行了解。

1.  **购买订阅**：在上述机场网站注册账号并购买套餐（推荐先买一个月试试）。
2.  **一键导入**：
    *   在机场官网首页找到“一键订阅”或“一键导入 Clash/Karing”。
    *   或者点击“复制订阅链接”，然后在软件配置中选择“从 URL 导入”。
3.  **开启使用**：
    *   在软件中选中刚刚导入的配置文件。
    *   点击“连接”或开启“系统代理”开关。
    *   连接后测试常用网站和 DNS 是否正常。
